# Online Quiz Website

## Overview
This project is a simple **Online Quiz Web Application** developed using **HTML, CSS, and JavaScript**. The application presents multiple-choice questions to the user and calculates the score based on the selected answers.

The purpose of this project is to demonstrate basic web development concepts and interactive quiz functionality.

---

## Features
- Multiple-choice quiz questions
- Automatic navigation to the next question
- Score calculation
- Final result display
- Simple and user-friendly interface

---

## Technologies Used
- HTML – Used to create the structure of the webpage  
- CSS – Used to style and design the interface  
- JavaScript – Used to implement quiz logic and score calculation  

---

## How the Project Works
1. The system displays a quiz question with four answer options.
2. The user selects one of the answers.
3. The program checks whether the selected answer is correct.
4. The quiz automatically moves to the next question.
5. After all questions are completed, the system displays the final score.

---

## Applications
This project can be used for:
- Online quiz practice
- Student knowledge testing
- Simple online examination systems
- Educational training programs

---

## How to Run the Project
1. Download or clone this repository.
2. Open the **index.html** file in any web browser.
3. Start answering the quiz questions.
4. The final score will be displayed after completing the quiz.
